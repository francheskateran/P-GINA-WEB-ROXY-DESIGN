# PAGINA-WEB-ROXY-DESIGN
La presente plataforma web surge de la necesidad de modernizar y digitalizar el emprendimiento de bisutería Roxy Design (fundado en 2016), el cual presentaba limitaciones de crecimiento debido a la falta de una plataforma propia. Tras un diagnóstico situacional, se identificó que la dependencia exclusiva de redes sociales y ferias físicas que vulneran la sostenibilidad del negocio ante crisis externas. La investigación propone el diseño y documentación de una página web de ventas personalizada. El objetivo principal es centralizar la gestión de inventario, optimizar el proceso de compra y aumentar la visibilidad de la marca. Mediante el uso de herramientas de software accesible, se ofrece una solución técnica que supera las barreras económicas y la brecha digital de las propietarias, beneficiando tanto al negocio como a su comunidad de clientes en Caracas.
